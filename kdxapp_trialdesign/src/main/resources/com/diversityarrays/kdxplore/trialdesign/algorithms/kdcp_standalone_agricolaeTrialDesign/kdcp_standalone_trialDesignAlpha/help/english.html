This experiment can be with the following criteria:
  • Have replicated plantings of each treatment n_rep times
  • Have a specific block size
  • Sufficiently many environemnts (blocks) to test on (n_treatments / block_size * n_rep)

The image at the top shows a layout of an experiment with a block size of 3 and 2 replications, totaling 6 blocks.

RULES

There must be more plots per block then block_size, i.e. block_size <= rows * cols

The number of treatments must be the square of an integer

Additionally, the following rules must be satified:

Given: s=length(treatmentnames)/block_size;
CONDITION RULE
nrep=2 block_size <= s
nrep=3,s is odd block_size<=s
nrep=3,s is even block_size<=s-1
nrep=4, s odd block_size <= s
